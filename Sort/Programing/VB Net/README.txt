Microsoft® Visual Basic® for Windows® Phone Developer Tools CTP September 2010

Prerequisites:

- Windows Phone Developer Tools RTW
- Visual Studio 2010 Professional or higher

Known Issues:

1. XNA projects. Visual Basic is not currently supported for XNA. Developers using the CTP can create Visual Basic Silverlight Windows Phone applications; however, there are no Visual Basic Windows Phone project templates available for XNA at this time.

2. Integration with Expression Blend for Windows Phone. Visual Basic Windows Phone applications are not currently supported in Expression Blend for Windows Phone.
